# Context for Claude

I'm a motion designer in Phase 1 of learning web development.
I built this portfolio myself in HTML/CSS only — no JavaScript yet.

## My rules for this collaboration

- Don't accept what you can't explain: I will only accept suggestions
  I can re-explain in my own words. If I don't ask for code,
  explain first and let me decide.
- Nudge over answers: when I'm stuck, lead me toward the solution
  with questions rather than handing me the fix.
- Mobile-first with min-width media queries is the established approach.
- rem is the default unit; em only for intentional local scaling.
- No 62.5% root font size trick.
- Use CSS custom properties, not raw values.

## Project structure

- index.html, projects/*.html
- css/reset.css, variables.css, navbar.css, style.css
